#  Bomberman

Bomberman remake (for now only core) written in SFML by **PiGames**

## How to play

### Basics
Rules are similar to standard rules: https://en.wikipedia.org/wiki/Bomberman

### Controlls
Use your *movement keys* to move and *action key* to place a bomb

#### Player 1
- [arrows] movement 
- [spacebar] action key

#### Player 2
- [WASD] movement
- [LCtrl] action key
